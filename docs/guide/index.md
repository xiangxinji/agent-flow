# 介绍

欢迎使用 AgentFlow - 一个强大且灵活的工作流引擎。

## 什么是 AgentFlow？

AgentFlow 是一个基于节点的工作流执行系统，专为构建复杂的 AI 应用和自动化流程而设计。它提供了直观的图形化界面和强大的编程接口，让开发者能够轻松创建、管理和执行复杂的工作流。

## 核心特性

### 🚀 高性能执行引擎

- **DAG 调度**：基于有向无环图的工作流调度，确保任务按正确的顺序执行
- **并行执行**：支持多个独立任务的并行处理，大幅提升执行效率
- **依赖管理**：自动处理节点间的依赖关系，确保数据正确流转

### 🎯 灵活的节点系统

- **Agent 节点**：与 AI 模型交互，执行智能推理任务
- **Function Call 节点**：调用预定义函数，执行特定操作
- **Branch 节点**：条件分支，根据不同条件执行不同路径
- **Iterator 节点**：遍历数组数据，批量处理任务
- **Parallel 节点**：并行执行多个任务分支

### 🔧 强大的状态管理

- **状态追踪**：实时追踪工作流执行状态
- **数据流转**：支持节点间的数据传递和转换
- **历史记录**：完整的工作流执行历史，便于审计和调试

### 🎨 可视化设计

- **图形化界面**：直观的拖拽式工作流设计
- **实时预览**：即时查看工作流结构和执行状态
- **调试工具**：强大的调试功能，快速定位问题

### 🔌 插件化架构

- **自定义节点**：轻松创建自定义节点类型
- **函数扩展**：灵活的函数注册和调用机制
- **第三方集成**：支持与各种第三方服务集成

## 适用场景

AgentFlow 适用于多种场景：

- **AI 应用开发**：构建复杂的 AI Agent 应用
- **数据处理流程**：创建 ETL 数据处理管道
- **业务流程自动化**：自动化各种业务流程
- **API 编排**：组合多个 API 调用形成复杂业务逻辑
- **智能客服系统**：构建多轮对话的智能客服
- **内容生成流程**：自动化内容创作和发布流程

## 技术栈

AgentFlow 基于现代技术栈构建：

- **TypeScript**：类型安全的开发体验
- **Mastra**：强大的 AI Agent 框架
- **Hono**：高性能的 Web 框架
- **Zod**：运行时类型验证
